# Download Export Branding Rules

## Mandatory Rule

Whenever we add any `Download` option for `PDF`, `Excel`, or similar export formats, the exported file must always include:

1. `School Logo`
2. `School Name`
3. `School Contact Information`
4. `School Address`

These details are compulsory for every school-facing export.

## Required School Information

At minimum, every export must contain:

- `School Logo`
- `School Name`
- `School Email`
- `School Phone`
- `School Address`

If available, also include:

- `School Website`
- `Academic Year`
- `Generated Date`

## Applies To

This rule applies to:

- Student reports
- Staff reports
- Fee receipts
- Salary slips
- Attendance exports
- Exam result exports
- Visitor exports
- Inventory exports
- Finance reports
- Any future PDF download
- Any future Excel download

## PDF Requirement

Every PDF export should have a proper letterhead/header section with:

- logo on top or top-left
- school name as the main heading
- address, phone, email, and website below the heading

## Excel Requirement

Every Excel export should reserve the top rows for school branding and information before the table data starts.

Recommended order:

1. School Name
2. School Address
3. School Phone
4. School Email
5. School Website
6. Report Title
7. Generated Date

## Developer Checklist

Before completing any new export feature, verify:

- school logo is included
- school information is included
- report title is included
- generated date is included
- export is readable when printed or shared

## Non-Negotiable

No PDF or Excel download should be added without school branding and school information.
